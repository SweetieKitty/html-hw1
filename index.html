Інформація про використані теги

Тег HTML Пояснення
"title" Задає заголовок сторінки, який відображається на вкладці браузера.
"h1 - h6" Заголовки різного рівня, від найбільшого (h1) до найменшого (h6).
"ul, li" Ненумерований список і його елементи.
"a" Тег для створення гіперпосилань.
"img" Тег для вставки зображень.
"p" Тег для абзаців тексту.
"table, thead, tbody, tfoot" Теги для створення таблиць і їх структурних частин.
"tr, th, td" Теги для створення рядків і комірок таблиці.
